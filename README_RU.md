# UEFA Champions League - Таблица турнира

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd La-Masia-master
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

### 2. Настройка API ключа

```bash
# Windows CMD
set RAPIDAPI_KEY=your_api_key_here

# Windows PowerShell
$env:RAPIDAPI_KEY="your_api_key_here"

# Linux/Mac
export RAPIDAPI_KEY="your_api_key_here"
```

### 3. Запуск приложения

```bash
python ucl_project/main.py
```

## 📊 Возможности

### ✅ Реализовано

- **Таблица турнира UCL 2024/25**
  - Реальные данные с SofaScore API
  - Автоматическое обновление каждые 1 час (кэш)

- **Полная статистика команд**
  - PLD (матчей сыграно)
  - W/D/L (победы/ничьи/поражения)
  - GF/GA/GD (голы забитые/пропущенные/разница)
  - PTS (очки)

- **Форма команд (последние 5 матчей UCL)**
  - 🟢 Зелёный = Победа (W)
  - ⚪ Серый = Ничья (D)
  - 🔴 Красный = Поражение (L)

- **Автоматический fallback**
  - Если сезон 2025/26 недоступен → показывает 2024/25
  - Если API недоступен (rate limit) → показывает mock данные
  - Если данные в кэше → показывает из кэша (1 час)

- **Визуальные индикаторы**
  - Зоны квалификации (цветные полосы)
  - Метка времени последнего обновления
  - Предупреждения при fallback/mock данных

## 🛠️ Решение проблем

### ❌ Ошибка: "429 Too Many Requests"

**Причина:** Превышен лимит запросов RapidAPI (бесплатный план: ~500 req/day)

**Решения:**

#### Вариант 1: Подождать (Рекомендуется)
```bash
# API лимиты обновляются каждый час
# Подождите 1 час и запустите снова
```

#### Вариант 2: Mock данные (Автоматически)
```bash
# Приложение автоматически загрузит mock данные
# Вы увидите:
# "League Phase 2024/25 (Mock Data - API Rate Limited)"
```

#### Вариант 3: Использовать кэш
```bash
# Если данные были загружены ранее,
# они сохранены в .cache/ на 1 час
# Приложение автоматически использует кэш
```

#### Вариант 4: Новый API ключ
```bash
# Создайте новый аккаунт на RapidAPI
# или используйте другой API ключ
set RAPIDAPI_KEY=new_api_key_here
```

### ❌ Форма команд пустая (показывает "-")

**Причины:**
1. **Rate limit** - слишком много запросов к API
2. **Команда не найдена** - ID команды отсутствует в маппинге
3. **Ошибка API** - временная проблема с SofaScore

**Решение:**
```bash
# Форма загружается отдельными запросами
# Если rate limit - подождите 1 час
# Или используйте mock данные (автоматически)
```

## 📝 Технические детали

### API Endpoints используемые:

1. **Standings:**
   ```
   GET /api/v1/unique-tournament/7/season/{season}/standings/total
   ```

2. **Team Events (для формы):**
   ```
   GET /api/v1/team/{team_id}/events/last/0
   ```

### Rate Limiting:

- **Задержка:** 0.5s каждые 5 команд
- **Кэш:** 1 час
- **Retry:** Нет (сразу fallback на mock)

### Производительность:

- **Standings:** 1 запрос (~0.5s)
- **Form (36 команд):** 36 запросов (~7s с задержками)
- **Total загрузка:** ~7-8 секунд
- **С кэшем:** ~0.1 секунда

## 🐛 Известные проблемы

1. **Форма не загружается для всех команд**
   - Причина: Rate limiting или неверный team_id
   - Решение: Добавить больше team IDs в api.py:311-347

2. **Unicode ошибки в консоли Windows**
   - Причина: Кодировка cp1251 не поддерживает некоторые символы
   - Решение: Использовать Windows Terminal или игнорировать

3. **Первый запуск медленный**
   - Причина: Загрузка формы всех команд (36 запросов)
   - Решение: Использовать кэш (автоматически после первого запуска)
